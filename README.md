# SiteLense

SiteLense is a high-performance website scanner designed to provide instant insights into web performance, SEO, and accessibility. Built with the latest Next.js features, it offers a seamless experience for anyone looking to audit and optimize their web presence.

## 💡 Why SiteLense?

- **Instant Performance Check:** See exactly how fast your site loads for real users.
- **Easy-to-Read Reports:** No complex jargon—get clear scores for speed, SEO, and accessibility.
- **Smart Fix Suggestions:** Get a simple list of what to fix first to improve your site's ranking.
- **Beautiful Dashboard:** Track your site's health through a clean, distraction-free interface.
- **Reliable Results:** Powered by Google's industry-standard Lighthouse engine for data you can trust.
